How to Play

Master the game with these simple steps

1. Start Your Challenge

Begin with level 1, featuring a single-digit number. Each subsequent level adds one digit to the challenge.

2. Memorize the Number

You have 5 seconds to memorize the displayed number. Focus and use memory techniques to enhance retention.

3. Input Your Answer

After the timer ends, input the number you memorized. Accuracy is key to advancing to the next level.

4. Track Your Progress

Each correct answer earns points based on the level difficulty. Level 1 = 5 points, Level 2 = 10 points, up to Level 20 = 100 points.
